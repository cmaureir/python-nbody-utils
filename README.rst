python-nbody-utils
==================

A set of different utilities for N-body algorithms.


Features
---------

 * Kinetic and Potential energy calculation.
 * Half-mass radius.
 * Core radius.
 * Relaxation time.
 * Crossing time.
 * Core collapse time.

Future work
------------

 * Conversion between N-body units and Physical units.
 * ...


Usage
------


INPUT file format
~~~~~~~~~~~~~~~~~~

Format:
::

    <mass> <rx> <ry> <rz> <vx> <vy> <vz>
